Qual das opções abaixo possui o código completo da função exibeTexto? Mas é preciso que o código funcione conforme esperado, que é exibir um alerta para o usuário. Quando a função exibeTexto for chamada, ela passará como parâmetro da função alert o valor input.value. Não podemos passar simplesmente input, porque o input corresponde a TAG e queremos dessa TAG seu valor (value). Por isso passamos input.value.
